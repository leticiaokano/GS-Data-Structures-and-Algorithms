Explicação da Lógica Usada:

Para o programa, foi usado a estrutura switch no menu, que permite ao usuário inserir dados, visualizar informações e executar análises. As verificações são feitas através de estruturas condicionais (if), que identifica situações de superaquecimento, baixo nível de energia e falhas de comunicação. O programa permanece em execução utilizando um laço de repetição até que o usuário escolha a opção de encerramento.
